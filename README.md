# HealthTrack

A simple accessible health tracking web app.

## Run locally

```bash
python server.py
```

Then open:

- http://127.0.0.1:8080/welcome.html

## Deploy to Vercel

1. Create a GitHub repository for this folder.
2. Push the project to GitHub.
3. In Vercel, import the repository.
4. Use the default settings.
5. Deploy.

The project is ready for static hosting on Vercel.
